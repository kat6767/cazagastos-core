# ISO/IEC 25010

## Características consideradas

### Usabilidad
- Registro rápido de gastos.
- Facilidad de aprendizaje.

### Fiabilidad
- Conservación de información en modo offline.

### Seguridad
- Bloqueo de accesos no autorizados.

### Eficiencia de desempeño
- Procesamiento de tickets en menos de 3 segundos.

### Mantenibilidad
- Complejidad ciclomática ≤ 10.
- Uso de revisión de código.
