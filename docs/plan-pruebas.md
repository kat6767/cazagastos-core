# Plan de Pruebas

## Niveles de prueba

### Pruebas unitarias
Validación de funciones individuales.

### Pruebas de integración
Validación de interacción entre módulos.

### Pruebas de caja negra
Evaluación de entradas y salidas del sistema.

### Pruebas de usabilidad
Medición de facilidad de uso y tiempos de ejecución.

### Revisión de código
Verificación de mantenibilidad y estándares.
