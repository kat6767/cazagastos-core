# Requisitos de CazaGastos

## Requisitos funcionales

### RF-01
El sistema debe permitir subir fotografías de tickets o facturas.

### RF-02
El sistema debe permitir guardar registros de gastos sin conexión a internet.

### RF-03
El sistema debe permitir registrar el primer gasto en un máximo de 15 segundos.

### RF-04
El sistema debe denegar el acceso con credenciales inválidas.

### RF-05
El sistema debe validar que el monto ingresado sea un valor numérico positivo.

## Requisitos no funcionales

- Tiempo de respuesta para carga de tickets ≤ 3 segundos.
- Complejidad ciclomática máxima de 10 por función.
- Conservación de datos en modo offline.
- Interfaz orientada a dispositivos móviles.
